# 图形推理题 - 规则说明

## 基础规则

### 位置排列移动 (Position Permutation)
- 每道题有 3 行，每行有 3 个田字格（2x2 格子，4个位置：左上TL、右上TR、左下BL、右下BR）
- 每行使用 4 个不同的 emoji，不同行之间 emoji 互不相同
- 核心规律：**所有** emoji 按照固定的 4-cycle 排列规则在 3 个格子间移动位置
  - Grid2 = P(Grid1), Grid3 = P(Grid2)，其中 P 是同一个位置置换
- 3 行使用相同的置换规则，但每行的 emoji 初始排列随机
- 第 3 行的第 3 个格子为待求的答案，由玩家填写

## 额外加难规则（视觉变换）

每道题随机选取 **1~2 个**不同的加难规则，分别应用到不同的 emoji 上。
有加难规则的 emoji 行为：

- **所有格子中都跟着位置排列规则正常移动**（和普通 emoji 一样）
- **在 Grid 1 和 Grid 2 中外观完全一样**（正常大小、正常方向）
- **只在 Grid 3 中外观发生变化**（缩小/镜像/旋转）

### 规则 1：缩小 (Shrink)
- Grid 1、Grid 2 中大小正常
- Grid 3 中变小（缩放约 55%）

### 规则 2：左右镜像 (Mirror-X)
- Grid 1、Grid 2 中正常显示
- Grid 3 中左右镜像对称（水平翻转）

### 规则 3：顺时针旋转 90° (Rotate-90)
- Grid 1、Grid 2 中正常显示
- Grid 3 中顺时针旋转 90 度

### 规则 4：上下镜像 (Mirror-Y)
- Grid 1、Grid 2 中正常显示
- Grid 3 中上下镜像对称（垂直翻转）

## 规则组合方式

每道题随机选取：
1. **一个 4-cycle 位置置换**（6 种中随机选一种，所有 emoji 统一使用）
2. **1~2 个视觉变换规则**（从4种中随机选，不重复），分别分配给不同的 emoji

玩家需要同时观察：
- 每个 emoji 的移动规律（位置排列）
- 哪些 emoji 在 Grid 3 中外观发生了变化（加难规则）

## 答案选项设计

每道题有 8 个选项，来自第 3 行的 4 个 emoji，每个 emoji 出现两次：

1. **正确版本**：emoji 以正确的视觉状态呈现
   - 有变换规则的 emoji → 显示变换后的样子（缩小/镜像/旋转）
   - 无变换规则的 emoji → 显示正常样子
2. **干扰版本**：同一个 emoji 以错误的视觉状态呈现
   - 有变换规则的 emoji → 干扰项显示正常（未变换）的样子
   - 无变换规则的 emoji → 干扰项显示随机变换后的样子

玩家需要同时判断：
- 哪个 emoji 放在哪个位置（位置排列规律）
- 哪个视觉版本是正确的（变换规律）

## Emoji 选择要求

- 不使用对称图形的 emoji（圆形、方形、爱心、星星等）
- 优先选择有明确方向性的 emoji（动物、乐器、交通工具、食物等）
- 确保旋转或镜像后能明显看出视觉差异
